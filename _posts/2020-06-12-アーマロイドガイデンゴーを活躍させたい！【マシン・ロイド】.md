## 解説

主役のアーマロイドガイデンゴーは二通りの使い方があります！まずはマシンナーズ・フォートレスのコストにする、ちょっとかわいそうな使い方。アーマロイドガイデンゴーは★8なので、一枚でマシンナーズ・フォートレスのコストにできます。エクスプレスロイドで二枚サルベージできればマシンナーズ・フォートレスを2回召喚できます！

{% highlight plain text %}
《アーマロイドガイデンゴー/Armoroid》
効果モンスター
星８/地属性/機械族/攻2700/守2000
(1)：「ロイド」モンスターをリリースしてこのカードがアドバンス召喚に成功した場合に発動する。
フィールドの魔法・罠カードを全て除外する。{% endhighlight %}

{% highlight plain text %}
《マシンナーズ・フォートレス/Machina Fortress》
効果モンスター
星７/地属性/機械族/攻2500/守1600
このカードは手札の機械族モンスターを
レベルの合計が８以上になるように捨てて、
手札または墓地から特殊召喚する事ができる。
このカードが戦闘によって破壊され墓地へ送られた時、
相手フィールド上に存在するカード１枚を選択して破壊する。
また、自分フィールド上に表側表示で存在する
このカードが相手の効果モンスターの効果の対象になった時、
相手の手札を確認して１枚捨てる。{% endhighlight %}

{% highlight plain text %}
《エクスプレスロイド/Expressroid》
効果モンスター
星４/地属性/機械族/攻 400/守1600
(1)：このカードの召喚・反転召喚・特殊召喚に成功した時、
「エクスプレスロイド」以外の自分の墓地の「ロイド」モンスター２体を対象として発動できる。
そのモンスターを手札に加える。{% endhighlight %}

もう一つの使い方は、普通に召喚して疑似大嵐を発動する使い方です。シャインエンジェルでユーフォロイドを出して、黄泉ガエルorレベル・スティーラーを添えてアドバンス召喚！！大嵐ドッカーン！！

{% highlight plain text %}
《ユーフォロイド/UFOroid》
効果モンスター
星６/光属性/機械族/攻1200/守1200
このカードが戦闘によって破壊され墓地へ送られた時、
デッキから攻撃力１５００以下の機械族モンスター１体を
表側攻撃表示で特殊召喚する事ができる。
その後デッキをシャッフルする{% endhighlight %}



ジェネクスのエンジンが絶妙に噛み合っているところもGood！

- ウンディーネで黄泉ガエルを墓地に落とせる

- 手札に来たジェネクス・コントローラーはマシンナーズ・フォートレスのコストに出来る

- ユーフォロイド＋レベル・スティーラー＋ジェネクス・コントローラーでトリシューラを出せる

![]({{site.baseurl}}\assets/6dcb32a026e3403b9f7134e16fce1616/.png)

黄泉ガエルが入っているので罠カードは入っていません。代わりに入っている防御札もシナジー抜群です！OCG未実装のカイトロイドはデュエルリンクスでは最速制限リスト入りした強カード。黄泉帝で使用率が高い類似カードのバトル・フェーダーとの差別化は

- 機械族だからマシンナーズ・フォートレスのコストに使える

- コストにした後、墓地でも効果を発動できる

- 「ロイド」モンスターなのでエクスプレスロイドでサルベージできる

です！

{% highlight plain text %}
《カイトロイド/Kiteroid》
効果モンスター
星１/風属性/機械族/攻200/守00
相手モンスターが直接攻撃した場合、そのダメージ計算時に、このカードを手札から捨てて発動できる。
その戦闘によって発生する自分への戦闘ダメージを0にする。
また、相手モンスターが直接攻撃した場合、そのダメージ計算時に、
墓地のこのカードを除外して発動できる。
その戦闘によって発生する自分への戦闘ダメージを0にする。{% endhighlight %}



後はエフェクト・ヴェーラー。シャインエンジェルとワン・フォー・ワンに対応しているので、シンクロ召喚にも繋げやすく、こちらもシナジー抜群です。
